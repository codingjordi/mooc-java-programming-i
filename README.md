## 🛠️ Tech Stack & Tools

* **Language:** Java 11
* **IDE:** TMCBeans (NetBeans with the Test My Code plugin)
* **Testing:** TMC (Test My Code) framework for automated local and server-side validation.

## 📁 Project Structure

Each folder represents a specific exercise from the course. The internal structure follows the standard Maven/NetBeans layout:
* `src/main/java`: Contains the source code for the solution.
* `src/test/java`: Contains the automated unit tests provided by the university.

## 💻 How to Run the Exercises

If you have **TMCBeans** installed:
1. Clone this repository: `git clone https://github.com/your-username/your-repo-name.git`
2. Open TMCBeans.
3. Go to `File -> Open Project`.
4. Navigate to the specific exercise folder and open it.
5. Click the **TMC "Test" button** to run the automated tests.
